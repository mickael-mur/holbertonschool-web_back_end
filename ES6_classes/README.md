# ES6 Classes

This project covers ES6 classes in JavaScript, including:
- Class declaration and instantiation
- Getters and setters
- Static methods
- Class inheritance
- Abstract classes
- Type validation in constructors
- Symbol usage for advanced features

## Files

- `0-classroom.js` - Basic class implementation
- `1-make_classrooms.js` - Function returning class instances
- `2-hbtn_course.js` - Class with getters, setters, and type validation
- `3-currency.js` - Class with methods
- `4-pricing.js` - Class with static methods
- `5-building.js` - Abstract class implementation
- `6-sky_high.js` - Class inheritance
- `7-airport.js` - Custom string representation
- `8-hbtn_class.js` - Type casting with classes
- `9-hoisting.js` - Fixing hoisting issues
- `10-car.js` - Cloning objects with inheritance

## Setup

Make sure you have Node.js and npm installed. The project uses the same configuration as ES6_basic (babel.config.js and package.json).
