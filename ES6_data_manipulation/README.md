# ES6 Data Manipulation

Ce projet porte sur la manipulation de données en JavaScript ES6, notamment :
- Les méthodes de tableau (map, filter, reduce)
- Les Typed Arrays
- Les structures de données Set et Map

## Exercices

0. **Basic list of objects** - Créer une fonction qui retourne un tableau d'objets étudiants
1. **More mapping** - Extraire les IDs avec map
2. **Filter** - Filtrer les étudiants par localisation
3. **Reduce** - Calculer la somme des IDs avec reduce
4. **Combine** - Combiner filter et map pour mettre à jour les notes
5. **Typed Arrays** - Créer un Int8TypedArray
6. **Set data structure** - Créer un Set depuis un tableau
7. **More set data structure** - Vérifier si des valeurs existent dans un Set
8. **Clean set** - Filtrer et formater les valeurs d'un Set
9. **Map data structure** - Créer une Map de courses
10. **More map data structure** - Mettre à jour les valeurs d'une Map
